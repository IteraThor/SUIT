# SUIT - Setup Utilities by IteraThor

SUIT is a graphical toolkit designed to simplify the deployment and management of an **Autodarts system** on Touchscreen running Linux systems, specifically optimized for Ubuntu and Wayland environments.

## Installation

The easiest way to install SUIT is to use the provided installation script:

```bash
git clone https://github.com/IteraThor/SUIT.git
cd SUIT
chmod +x install.sh
./install.sh
```


## Usage
Once installed, you can launch SUIT directly from your application menu or the desktop icon.
